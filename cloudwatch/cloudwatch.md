What is cloud watch:
===================

    Amazon CloudWatch is a monitoring and observability service designed for DevOps engineers, developers, 
    and IT managers using AWS. It collects monitoring data—logs, metrics, and events—from AWS resources (like EC2, RDS, Lambda)
     and on-premises servers to provide actionable insights, automated alerts, and real-time dashboards for troubleshooting and performance 
     optimization.
